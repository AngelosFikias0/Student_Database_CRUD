# Student Management System

This repository hosts a simple application for managing student data. It provides basic CRUD (Create, Read, Update, Delete) operations via a web interface.

## Features

- **Create Student:** Add new students with their name, email, and age.
- **Read Students:** View a list of all students currently stored in the database.
- **Update Student:** Modify existing student records by ID.
- **Delete Student:** Remove student records from the database.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

### Prerequisites

- Install XAMPP (or any other local server environment that supports PHP and MySQL).
- Clone this repository to your local machine.
